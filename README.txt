Η καταγραφή στο manager-log-file για τις εργασίες συγχρονισµού δεν έχει υλοποιηθεί.
Δεν καταγράφονται εγγραφές σε μορφή: [TIMESTAMP] [SOURCE_DIR] [TARGET_DIR] [THREAD_PID] [OPERATION] [RESULT] [DETAILS].

Το cancel command στον conosle δεν κάνει κατι ουσιόδες, απλός εκτυπώνει στην οθόνη το κατάλληλο μήνυμα.

Για να σταλθουν add commands απο το console: πρεπει να ειναι στην μορφη /source_dir@host:port /target@....
χωρις το '/' η εντολή δεν λειτουργεί. 

Για την αποθήκευση των καταλόγων εχω χρησιμοποιησει hash table.

Βοηθητικές functions που χρησιμοποιούνται υπάρχουν στο include/helper.h

Εκτός αυτών, η εργασία είναι σύμφωνη με τις απαιτήσεις της εκφώνηση.
